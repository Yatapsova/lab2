# FdF (Fil de Fer) - 3D Wireframe Viewer

Проект FdF - программа для отображения трехмерных каркасных моделей из файлов FDF и изображений.

## Описание

FdF (с французского "Fil de Fer" - "проволочный каркас") - это программа для визуализации 3D моделей в виде каркасных структур. Программа считывает данные о высотах из файлов и преобразует их в трехмерные проекции.

## Возможности

### Основная часть (fdf.py):
- Загрузка файлов формата .fdf
- Отображение 3D моделей в виде каркаса
- Вращение модели по трем осям
- Масштабирование и перемещение камеры
- Автоматическое вращение
- Отображение осей координат
- Цветовая градиентная карта по высоте

### Бонусная часть (fdf_bonus.py):
- Поддержка изображений (PNG, JPEG, BMP, TIFF)
- Три режима отрисовки: каркас, точки, заливка
- Сглаживание и анимация
- Более продвинутая система цветов
- Сортировка граней для правильного отображения
- Эффекты затухания

Управление

Основные клавиши:

· W/S - Вращение по оси X
· A/D - Вращение по оси Y
· Q/E - Вращение по оси Z
· +/- - Увеличение/уменьшение масштаба
· Стрелки - Перемещение камеры
· R - Сброс вида
· ESC - Выход

Дополнительные (в бонусной версии):

· 1 - Режим каркаса
· 2 - Режим точек
· 3 - Режим заливки
· T - Вкл/выкл авто-вращение
· G - Вкл/выкл сетку
· X - Вкл/выкл оси координат

Формат FDF файла

FDF файл представляет собой текстовый файл, где:

· Каждая строка - ряд точек по оси Y
· Каждое значение в строке - высота (Z координата)
· Значения разделяются пробелами
· Опционально можно указать цвет в формате: высота,цвет
· Цвет задается в hex формате: 0xRRGGBB или RRGGBB

Пример:


0 10 20
5 15 25
0 10 20


Пример с цветами:


0,0xFF0000 10,0x00FF00
20,0x0000FF 30,0xFFFF00


Поддерживаемые форматы изображений

Бонусная версия поддерживает:

· PNG (.png)
· JPEG/JPG (.jpg, .jpeg)
· BMP (.bmp)
· TIFF (.tiff, .tif)

Изображения конвертируются в grayscale, где яркость пикселя определяет высоту точки.
